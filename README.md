# weixin-xiaochengxu
微信小程序编程学习，一款专门学习微信小程序的小程序

## 不多说，先看效果图

![图片地址](https://github.com/qiushi123/weixin-xiaochengxu/blob/master/guide/%E9%A1%B5%E9%9D%A2.png?raw=true)

## 已实现功能
- 1，post请求json列表
- 2，首页列表
- 3，学习详情页
- 4，个人中心
- 5，登陆获取用户名和头像
- 6，openid的获取
- 7，你问我答模块
- 8，提问页面
- 9，我的提问列表页
- 10，评论功能

## 用到的技术
- 1，小程序云开发实现你问我答
- 2，借助小程序云开发数据库存储问答数据
- 3，借助阿里云服务器实现文章的存储
